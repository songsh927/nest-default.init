## Description   

NestJS project preset

## Default Packages

- typeorm   
- sequelize   
- mysql2   
- winston   
- nest-winston   
- dotenv   

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# watch mode
$ npm run debug

# production mode
$ npm run prod
```
